## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue. Instead, please report it via one of the following methods:

- **Email:** [mvdnbrk@protonmail.com](mailto:mvdnbrk@protonmail.com)
- **Twitter/X:** [@mvdnbrk](https://x.com/mvdnbrk)

Please include a detailed description of the vulnerability and steps to reproduce it. We aim to acknowledge all reports within 48 hours and provide a fix as quickly as possible.

### What to include in your report:
- A clear description of the issue.
- Impact: What could an attacker do?
- Reproduction steps or a Proof of Concept (PoC).
- Any suggested fixes (optional).
